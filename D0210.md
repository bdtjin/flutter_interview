# Oauth 2.0 이란 무엇인가요?

## OAuth 2.0
Open Authorization 2.0의 약자로 웹 및 애플리케이션 인증 및 권한 부여를 위한 개방형 표준 프로토콜입니다.
애플리케이션이 사용자의 리소스에 접근하기 위한 절차를 정의하고 서비스 제공자의 API를 사용할 수 있는 권한을 부여합니다.

## OAuth 2.0 구성

### 1. Resource Owner (자원 소유자)
사용자 본인

### 2. Client (클라이언트)
사용자의 리소스에 접근하려는 애플리케이션

### 3. Authorization Server (인증 서버)
사용자의 신원을 확인하고 클라이언트에게 권한을 부여하는 서버

### 4. Resource Server (자원 서버)
사용자의 리소스를 저장하고 있는 서버

## OAuth 2.0 배경
앱에서 Naver의 정보를 가져오기 위해서는 Naver의 ID와 PW를 직접 입력받아서 앱에 저장해서 필요할 때마다 불러와서 사용
사용자/앱/Naver 모두 보안에 취약
 - 사용자: 앱에 Naver의 ID와 PW를 넘겨주는 것에 신뢰할 수 없음
 - 앱 : Naver의 ID와 PW를 저장해야하기 때문에 보안에 문제 생기는 경우 모든 책임을 져야 함
 - Naver : 앱을 신뢰할 수 없음
